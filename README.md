MiraiNime adalah aplikasi pencarian anime sederhana yang dibangun menggunakan **React**, **TypeScript**, dan **TailwindCSS**, dengan data diambil dari **Jikan API** (unofficial MyAnimeList API).  
Aplikasi ini memungkinkan pengguna untuk mencari anime berdasarkan judul dan menampilkan hasil berupa gambar, judul, serta skor rating anime secara real-time.

## Fitur Utama

-   **Pencarian Anime:** Cari anime favoritmu berdasarkan nama.
-   **Integrasi Jikan API:** Data diambil langsung dari MyAnimeList melalui API publik.
-   **UI Responsif & Modern:** Menggunakan TailwindCSS dengan desain profesional.
-   **Dibangun dengan TypeScript:** Kode lebih aman dan mudah dikelola.
-   **Komponen Terpisah:** Navbar, Hero Section, dan Search dibuat modular.

## Tech Stack

| Teknologi        | Deskripsi                                                 |
| ---------------- | --------------------------------------------------------- |
| **React**        | Library frontend untuk membangun UI interaktif            |
| **TypeScript**   | Superset JavaScript untuk type-safety dan maintainability |
| **TailwindCSS**  | Framework CSS utility-first untuk styling cepat           |
| **Jikan API v4** | REST API untuk mengambil data anime dari MyAnimeList      |

## Cara Menjalankan Proyek

### 1. Clone repository

```bash
git clone https://github.com/rizkicahyaa/mirai-nime.git
cd mirai-nime
```

### 2. Menginstall npm

```bash
npm install
```

### 3. Jalankan aplikasi

```bash
npm run dev
```
